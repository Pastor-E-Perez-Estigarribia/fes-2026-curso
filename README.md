## **📚 Estructura del Módulo**

Este repositorio sigue un enfoque bilingüe (R y Python) para el Análisis Exploratorio de Datos, Inferencia, Modelado Multivariado y Simulación.

### **Ejes Temáticos:**

1. Análisis Exploratorio de Datos (EDA) y Modelado Exploratorio.  
2. Distribuciones de Probabilidad.  
3. Inferencia Estadística y Pruebas de Hipótesis.  
4. Análisis Multivariado (GLM, PCA).  
5. Series de Tiempo.  
6. Simulación (Monte Carlo y Sistemas de Colas).
