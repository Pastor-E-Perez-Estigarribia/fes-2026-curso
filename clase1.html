Unidad 1

EDA Integral: De Tukey a Wickham

Fundamentos univariados y multivariados (Tukey) integrados con el modelado exploratorio moderno (Wickham & Grolemund).

1 Abordaje Conceptual Integrado

El Análisis Exploratorio de Datos (EDA) es el pilar de la ciencia de datos empírica. Evoluciona desde la descripción inicial de los datos hasta la generación de hipótesis complejas, combinando dos filosofías fundamentales:

1. El Enfoque Clásico (John Tukey, 1977)

Tukey concibió el EDA como un "trabajo de detective" basado en describir la realidad de los datos antes de asumir modelos. Se enfoca en:

  • EDA Univariado (Variación): Entender la distribución individual de cada variable. ¿Dónde está el centro (Media/Mediana)? ¿Cuál es la dispersión (Varianza/IQR)? ¿Hay asimetría?
  • EDA Bivariado y Multivariado (Covariación): Evaluar cómo interactúan las variables (Covarianza, Correlación) mediante diagramas de dispersión o matrices.
  • Detección de Anomalías (Outliers): Identificar valores atípicos que podrían ser errores de medición o descubrimientos invaluables (usando Boxplots y rangos intercuartílicos).

2. El Modelado Exploratorio (Wickham & Grolemund)

El libro R for Data Science lleva el EDA un paso más allá al proponer un estado mental iterativo donde los modelos matemáticos se usan como herramientas de exploración, no solo de predicción.

Separar la Señal del Ruido: Ajustamos un modelo simple para capturar la "señal" fuerte (la relación obvia descubierta en el EDA Bivariado). Una vez extraída esa señal, analizamos los residuos (el ruido sobrante) para cazar anomalías sutiles o patrones ocultos multivariados que antes estaban enmascarados.

2 Caja de Herramientas Matemáticas

Límites de Outliers (Tukey):

$$ L_{inf} = Q_1 - 1.5 \times IQR $$ $$ L_{sup} = Q_3 + 1.5 \times IQR $$

Valores fuera de estos límites en un boxplot se consideran atípicos.

Correlación de Pearson (\(r\)):

$$ r = \frac{\sum (x_i - \bar{x})(y_i - \bar{y})}{\sqrt{\sum (x_i - \bar{x})^2 \sum (y_i - \bar{y})^2}} $$

Mide la fuerza y dirección de la asociación lineal bivariada (-1 a 1).

Modelo Exploratorio (Señal):

$$ \hat{y}_i = \beta_0 + \beta_1 x_i $$

Predicción matemática de la tendencia principal (covariación).

Cálculo de Residuos (Ruido):

$$ e_i = y_i - \hat{y}_i $$

El valor real menos el predicho. Lo que el modelo exploratorio no logró explicar.

3 Aplicación Industrial: Fatiga de Materiales

El problema: En una línea de ensamblaje, analizamos la Vida Útil de una herramienta de corte (\(Y\)) respecto a la Velocidad del rotor (\(X\)).

Fase 1 (Tukey): El EDA Univariado muestra con un Boxplot que la vida útil tiene algunos outliers severos. El EDA Bivariado revela una correlación de \(r = -0.85\): a mayor velocidad, menor vida útil. Esta es nuestra señal obvia.

Fase 2 (Wickham): Queremos saber si influye el "Proveedor" del material (variable multivariada oculta). Ajustamos un modelo lineal Vida ~ Velocidad para eliminar la penalización obvia de la velocidad. Al graficar los residuos por Proveedor, notamos que el Proveedor A siempre tiene residuos positivos (sus piezas duran más de lo que predice la matemática para esa velocidad). Usamos el modelo exploratorio como "lupa" para descubrir un patrón multivariado.

R Implementación en R (Pipeline Completo)

Desde la exploración visual básica hasta el modelado exploratorio con modelr.

library(tidyverse)
library(modelr)

# 1. Datos simulados de la industria
set.seed(42)
df_maquinas <- tibble(
  velocidad = runif(100, 1000, 3000),
  proveedor = sample(c("A", "B"), 100, replace = TRUE),
  # El Proveedor A tiene un bonus oculto de +200h de vida
  vida_util = 5000 - (velocidad * 1.2) + ifelse(proveedor == "A", 200, 0) + rnorm(100, 0, 50)
)

# 2. Fase Tukey: EDA Bivariado Visual
plot_tukey <- ggplot(df_maquinas, aes(x = velocidad, y = vida_util)) +
  geom_point(alpha = 0.6) +
  geom_smooth(method = "lm", se = FALSE, color = "darkgray") +
  labs(title = "EDA Bivariado (La Señal)")

# 3. Fase Wickham: Modelado Exploratorio
modelo <- lm(vida_util ~ velocidad, data = df_maquinas)

df_maquinas_exp <- df_maquinas |>
  add_predictions(modelo) |>
  add_residuals(modelo)

# 4. Revelando el EDA Multivariado a través de Residuos
plot_wickham <- ggplot(df_maquinas_exp, aes(x = proveedor, y = resid, fill = proveedor)) +
  geom_boxplot() +
  geom_hline(yintercept = 0, color = "red", linetype = "dashed") +
  labs(title = "Análisis de Residuos: Descubrimiento Multivariado")

Py Implementación en Python (Pandas & Statsmodels)

Flujo de detección de outliers (Tukey) y extracción de residuos (Wickham).

import pandas as pd
import numpy as np
import seaborn as sns
import matplotlib.pyplot as plt
import statsmodels.formula.api as smf

# 1. Simulación (Mismos datos)
np.random.seed(42)
velocidad = np.random.uniform(1000, 3000, 100)
proveedor = np.random.choice(["A", "B"], 100)
bonus = np.where(proveedor == "A", 200, 0)
vida_util = 5000 - (velocidad * 1.2) + bonus + np.random.normal(0, 50, 100)

df_maquinas = pd.DataFrame({'velocidad': velocidad, 'proveedor': proveedor, 'vida_util': vida_util})

# 2. Detección de Outliers (Método IQR Tukey)
Q1 = df_maquinas['vida_util'].quantile(0.25)
Q3 = df_maquinas['vida_util'].quantile(0.75)
IQR = Q3 - Q1
outliers = df_maquinas[(df_maquinas['vida_util'] < (Q1 - 1.5 * IQR)) | (df_maquinas['vida_util'] > (Q3 + 1.5 * IQR))]

# 3. Modelado Exploratorio (Wickham)
modelo = smf.ols('vida_util ~ velocidad', data=df_maquinas).fit()
df_maquinas['residuos'] = modelo.resid

# 4. Visualización Conjunta
fig, axes = plt.subplots(1, 2, figsize=(14, 5))

# EDA Bivariado
sns.scatterplot(data=df_maquinas, x='velocidad', y='vida_util', ax=axes[0])
axes[0].set_title("EDA: Covariación Original")

# Análisis de Residuos Multivariado
sns.boxplot(data=df_maquinas, x='proveedor', y='residuos', ax=axes[1], palette="Set2")
axes[1].axhline(0, color='red', linestyle='--')
axes[1].set_title("Modelado Exploratorio: Residuos por Proveedor")

plt.tight_layout()
plt.show()
