1 Formalización Matemática de Series

Una Serie Temporal es una secuencia de variables aleatorias \(X_t\) indexadas en el tiempo \(t\). Para modelar estas series, introducimos dos conceptos esenciales: el Ruido Blanco y el Operador de Retardo (Lag).

Ruido Blanco (White Noise):

$$\epsilon_t \sim \mathcal{N}(0, \sigma^2)$$

Innovaciones puramente aleatorias. No hay autocorrelación: \(\text{Cov}(\epsilon_t, \epsilon_{t-k}) = 0\).

Operador de Retardo (B):

$$B X_t = X_{t-1} \quad \text{y} \quad B^k X_t = X_{t-k}$$

Permite formular ecuaciones complejas (como ARIMA) mediante polinomios algebraicos.

Estacionariedad (Stationarity)

Un modelo estocástico es estacionario de manera débil si sus propiedades estadísticas no varían a lo largo del tiempo. Es el supuesto base para los modelos clásicos (ARIMA).

  • Media constante: \(\mathbb{E}[X_t] = \mu\) para todo \(t\).
  • Varianza constante: \(\text{Var}(X_t) = \sigma^2\) independiente de \(t\) (homocedasticidad).
  • Autocovarianza depende del rezago: \(\gamma(h) = \text{Cov}(X_t, X_{t-h})\) depende solo de la distancia \(h\), no del tiempo \(t\).

2 Modelos Clásicos (ARIMA)

Proceso Autorregresivo AR(p)

El valor actual depende linealmente de sus \(p\) valores pasados.

$$X_t = \phi_1 X_{t-1} + \dots + \phi_p X_{t-p} + \epsilon_t$$

Proceso de Medias Móviles MA(q)

El valor actual depende linealmente de los \(q\) shocks/errores pasados.

$$X_t = \epsilon_t + \theta_1 \epsilon_{t-1} + \dots + \theta_q \epsilon_{t-q}$$
